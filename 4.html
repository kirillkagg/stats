Key Formulas — Lecture 4

Uniform PDF
f(x) = 1/(b−a), a ≤ x ≤ b
Uniform Mean & Variance
μ = (a+b)/2, σ² = (b−a)²/12
Continuous E(X)
E(X) = ∫ x · f(x) dx
Continuous Var(X)
Var(X) = E(X²) − [E(X)]²
Z Standardisation
Z = (X − μ) / σ
Linear function W = aX + b
E(W) = aμ+b, Var(W) = a²σ²
Normal approx. to Binomial
μ = np, σ² = np(1−p)
Valid when
np ≥ 5 and n(1−p) ≥ 5

Q1
Which of the following statements about a continuous random variable X is correct?
AThe probability that X takes any single specific value is always greater than zero.
BThe probability that X takes any single specific value is exactly zero; only intervals have positive probability.
CThe probability density function f(x) must always be less than or equal to 1 for all values of x.
DThe cumulative distribution function F(x) can take values greater than 1.
ENone of the above.
Correct answer: B

For any continuous RV, P(X = c) = 0 for any specific constant c, because the integral over a single point has zero width and therefore zero area. Only intervals carry positive probability: P(a < X < b) = ∫ab f(x)dx > 0.

Why others are wrong:
A — This describes discrete RVs.
C — f(x) can exceed 1 (e.g., f(x) = 2 on [0, 0.5] integrates to 1 and is valid).
D — F(x) = P(X ≤ x) is always between 0 and 1.
Q2
Which of the following is a valid probability density function on the interval 0 ≤ x ≤ 3?
Af(x) = 1/3 for 0 ≤ x ≤ 3, and 0 otherwise.
Bf(x) = x for 0 ≤ x ≤ 3, and 0 otherwise.
Cf(x) = −0.5x for 0 ≤ x ≤ 3, and 0 otherwise.
Df(x) = 2x for 0 ≤ x ≤ 3, and 0 otherwise.
ENone of the above.
Correct answer: A

A valid PDF needs: (i) f(x) ≥ 0, and (ii) total area = 1.

A: f(x) = 1/3 ≥ 0 ✓. ∫03(1/3)dx = 1 ✓ Valid.
B: ∫03 x dx = 9/2 = 4.5 ≠ 1 ✗
C: f(x) = −0.5x < 0 for x > 0 ✗ (density cannot be negative)
D: ∫03 2x dx = 9 ≠ 1 ✗
Q3
Consider the PDF f(x) = (3/8)x² for 0 ≤ x ≤ 2, 0 otherwise. What is E(X)?
A1.50
B1.20
C0.75
D2.00
ENone of the above.
Correct answer: A (E(X) = 1.50)

E(X) = ∫02 x · (3/8)x² dx = (3/8) ∫02 x³ dx = (3/8)[x&sup4;/4]02 = (3/8)(4) = 1.5

Validity check: ∫02(3/8)x²dx = (3/8)[x³/3]02 = (3/8)(8/3) = 1 ✓
Q4
Waiting time at a bus stop is uniformly distributed between 5 and 25 minutes. Which statement correctly gives the mean and variance?
AMean = 15 minutes, Variance = 33.33 minutes²
BMean = 15 minutes, Variance = 20 minutes²
CMean = 10 minutes, Variance = 33.33 minutes²
DMean = 15 minutes, Variance = 3.33 minutes²
ENone of the above.
Correct answer: A

Uniform(a=5, b=25):
Mean: μ = (5+25)/2 = 15 minutes
Variance: σ² = (25−5)²/12 = 400/12 = 33.33 minutes²
Standard deviation: σ = √33.33 ≈ 5.77 minutes
Q5
Machine off-line time is uniformly distributed between 20 and 80 minutes. What is the probability it is off-line for between 35 and 65 minutes?
A0.30
B0.40
C0.50
D0.60
ENone of the above.
Correct answer: C (0.50)

X ~ Uniform(20, 80). For uniform distributions: P(c ≤ X ≤ d) = (d−c)/(b−a)

P(35 ≤ X ≤ 65) = (65−35)/(80−20) = 30/60 = 0.50

The interval [35, 65] is exactly half the total range, so we expect probability = 0.5.
Q6
Let W = 3X + 5 where X has mean 4 and variance 9. What are E(W) and Var(W)?
AE(W) = 17, Var(W) = 81
BE(W) = 17, Var(W) = 27
CE(W) = 12, Var(W) = 81
DE(W) = 17, Var(W) = 32
ENone of the above.
Correct answer: A

Rules for linear functions W = aX + b:
E(W) = a·E(X) + b = 3(4) + 5 = 12 + 5 = 17
Var(W) = a²·Var(X) = 9 × 9 = 81

Key point: constants (the +5) shift the mean but do NOT affect variance. SD(W) = √81 = 9.
Q7
Let X ~ N(50, 100). Find P(X > 65) using standard normal tables.
A0.9332
B0.4332
C0.0668
D0.1336
ENone of the above.
Correct answer: C (0.0668)

X ~ N(μ=50, σ²=100), so σ = 10.

Standardise: Z = (65−50)/10 = 1.5
Table: Φ(1.5) = 0.9332
P(X > 65) = P(Z > 1.5) = 1 − 0.9332 = 0.0668

Only about 6.7% of values lie more than 1.5 standard deviations above the mean.
Q8
Let X ~ N(200, 225). Find P(185 < X < 215).
A0.3413
B0.6826
C0.9544
D0.4772
ENone of the above.
Correct answer: B (0.6826)

X ~ N(200, 225) so μ=200, σ=15.

Z1 = (185−200)/15 = −1.0    Z2 = (215−200)/15 = 1.0

P(−1 < Z < 1) = Φ(1) − Φ(−1) = 0.8413 − 0.1587 = 0.6826

This is the “68% rule”: approximately 68.26% of observations lie within ±1 standard deviation of the mean.
Q9
A fair coin is tossed 100 times. Using the normal approximation to the binomial, what is the probability of getting between 45 and 55 heads?
A0.3413
B0.5000
C0.6826
D0.9544
ENone of the above.
Correct answer: C (0.6826)

n=100, p=0.5. Validity: np=50≥5, n(1−p)=50≥5 ✓

μ = np = 50, σ² = np(1−p) = 25, σ = 5

Z1 = (45−50)/5 = −1.0,   Z2 = (55−50)/5 = 1.0
P = Φ(1) − Φ(−1) = 0.8413 − 0.1587 = 0.6826
Q10
Suppose X is normally distributed with mean 8 and standard deviation 5. Consider statements (i)–(iii):
(i) P(X < 8) = 0.5   (ii) P(X > 13) = P(X < 3)   (iii) P(6 < X < 10) ≈ 0.31
Which combination is correct?
A(i) only.
B(i) and (ii) only.
C(i), (ii), and (iii).
D(ii) and (iii) only.
ENone of the above.
Correct answer: C — all three are correct.

(i): P(X < μ) = 0.5 by symmetry of the normal distribution. ✓

(ii): P(X > 13) = P(Z > (13−8)/5) = P(Z > 1) = P(Z < −1) = P(X < 3). By symmetry these are equal. ✓

(iii): Z1 = (6−8)/5 = −0.4, Z2 = (10−8)/5 = 0.4.
P = Φ(0.4)−Φ(−0.4) = 0.6554−0.3446 = 0.3108 ≈ 0.31 ✓

Q11 Probability Density Functions — E(X), Var(X) and Probabilities 25% of exam
Consider the following probability density function:

f(x) = (2/9) x(3 − x)   for 0 ≤ x ≤ 3,   and 0 otherwise.
(a)Verify that f(x) is a valid probability density function. 20 marks

Worked Solution

Condition (i): f(x) ≥ 0. On [0,3]: x≥0 and (3−x)≥0, so f(x)=(2/9)x(3−x)≥0. ✓
Condition (ii): Total area = 1. ∫03(2/9)x(3−x)dx = (2/9)∫03(3x−x²)dx
= (2/9)[3x²/2 − x³/3]03 = (2/9)[(27/2) − (9)] = (2/9)(13.5−9) = (2/9)(4.5) = 1 ✓
Both conditions satisfied ⇒ f(x) is a valid PDF.
(b)Derive E(X). 25 marks

Worked Solution

E(X) = ∫03 x·(2/9)x(3−x)dx = (2/9)∫03(3x²−x³)dx
= (2/9)[x³ − x&sup4;/4]03 = (2/9)[27 − 81/4] = (2/9)(27−20.25)
= (2/9)(6.75) = 13.5/9 = E(X) = 1.5
(c)Derive Var(X). 25 marks

Worked Solution

Use Var(X) = E(X²) − [E(X)]²
E(X²) = (2/9)∫03x²·x(3−x)dx = (2/9)∫03(3x³−x&sup4;)dx
= (2/9)[3x&sup4;/4 − x&sup5;/5]03 = (2/9)[(243/4)−(243/5)] = (2/9)(243)(1/20) = 486/180 = 2.70
Var(X) = 2.70 − (1.5)² = 2.70 − 2.25 = 0.45
(d)Calculate P(1 ≤ X ≤ 2). 30 marks

Worked Solution

P(1≤X≤2) = (2/9)∫12(3x−x²)dx = (2/9)[3x²/2 − x³/3]12
At x=2: 6 − 8/3 = 10/3.   At x=1: 3/2 − 1/3 = 7/6
= (2/9)(10/3 − 7/6) = (2/9)(20/6 − 7/6) = (2/9)(13/6) = 26/54 = 13/27 ≈ 0.48
Q12 The Uniform Distribution — Properties and Applications 25% of exam
The time between customer arrivals at a bank (in minutes) follows a uniform distribution between 2 minutes and 10 minutes. Let X denote the inter-arrival time.
(a)State the key properties that define a uniform distribution. Write down f(x) for this problem. 25 marks

Worked Solution

Properties: (1) All outcomes in [a,b] are equally likely. (2) The probability of an event is proportional to interval length. (3) The PDF is constant (flat/rectangular shape).
PDF: f(x) = 1/(b−a) = 1/(10−2) = 1/8 = 0.125, for 2 ≤ x ≤ 10, and 0 otherwise.
Shape: A flat horizontal line at height 0.125. Total area = 0.125 × 8 = 1 ✓
(b)Calculate the mean and variance. 25 marks

Worked Solution

Mean: μ = (a+b)/2 = (2+10)/2 = 6 minutes
Variance: σ² = (b−a)²/12 = (10−2)²/12 = 64/12 ≈ 5.33 minutes²
Standard deviation: σ = √(16/3) ≈ 2.31 minutes
(c)Find P(X > 7) and interpret the result in context. 25 marks

Worked Solution

P(X > 7) = (10−7)/(10−2) = 3/8 = 0.375
Interpretation: There is a 37.5% probability that more than 7 minutes elapse between successive customer arrivals. In roughly 3 out of every 8 gaps, the bank teller faces an idle period exceeding 7 minutes.
(d)Find P(3 < X < 8). What proportion of arrival gaps fall within one standard deviation of the mean? Compare briefly with the normal distribution. 25 marks

Worked Solution

P(3 < X < 8) = (8−3)/(10−2) = 5/8 = 0.625
Within ±1 SD: μ ± σ = 6 ± 2.31 = (3.69, 8.31)
P(3.69 < X < 8.31) = (8.31−3.69)/8 = 4.62/8 ≈ 0.578
Comparison: For a normal distribution P(μ±σ) ≈ 68.3%. The uniform gives ≈57.8%, which is less concentrated around the mean — reflecting its flat shape with equal probability everywhere in [a,b].
Q13 The Normal Distribution — Z-Scores and Probabilities 25% of exam
Annual salaries at a firm are normally distributed with mean £35,000 and standard deviation £5,000. Let X denote an employee’s annual salary.
(a)State four key properties of the normal distribution. Why is it important in economics and statistics? 20 marks

Worked Solution — Four Properties

1. Symmetric: Mean = median = mode; the distribution is a mirror image about μ.
2. Bell-shaped: Probability is highest at the mean and tapers symmetrically in both directions.
3. Defined by μ and σ²: Two parameters completely characterise the distribution.
4. Asymptotic tails: Tails approach but never reach zero; the range is (−∞, +∞).
Importance: The Central Limit Theorem guarantees that sample means converge to normality for large n, regardless of population shape. This underpins confidence intervals, hypothesis testing, and regression inference.
(b)Find P(X > £42,000). 25 marks

Worked Solution

Z = (42,000 − 35,000)/5,000 = 7,000/5,000 = 1.40
P(X > 42,000) = P(Z > 1.40) = 1 − Φ(1.40) = 1 − 0.9192 = 0.0808
Interpretation: Approximately 8.1% of employees earn more than £42,000 per year.
(c)Find P(£28,000 < X < £40,000). 30 marks

Worked Solution

Z1 = (28,000−35,000)/5,000 = −1.40
Z2 = (40,000−35,000)/5,000 = 1.00
P = Φ(1.00) − Φ(−1.40) = 0.8413 − 0.0808 = 0.7605
Interpretation: About 76% of employees earn between £28,000 and £40,000.
(d)What salary level is exceeded by only 10% of employees? (Find the 90th percentile.) 25 marks

Worked Solution

We need x: P(X > x) = 0.10 ⇒ P(X ≤ x) = 0.90
From Z-table: Φ(z) = 0.90 ⇒ z = 1.2816
Reverse standardise: x = μ + z·σ = 35,000 + 1.2816 × 5,000 = £41,408
Interpretation: Only the top 10% of employees earn more than £41,408 per year.
Q14 Normal Approximation to the Binomial Distribution 25% of exam
A manufacturing process produces items where 30% are defective. In a batch of 150 items, let X denote the number of defective items.
(a)State the conditions under which the normal approximation to the binomial is valid. Verify them here, and state the parameters of the approximating normal distribution. 20 marks

Worked Solution

Conditions: np ≥ 5 AND n(1−p) ≥ 5.
Verification: np = 150×0.30 = 45 ≥ 5 ✓   n(1−p) = 150×0.70 = 105 ≥ 5 ✓
Parameters: μ = np = 45, σ² = np(1−p) = 150(0.3)(0.7) = 31.50, σ = √31.5 ≈ 5.612
(b)Using the normal approximation, find P(X ≤ 40) and interpret your result. 35 marks

Worked Solution

Z = (40 − 45)/5.612 = −5/5.612 ≈ −0.89
P(X ≤ 40) ≈ P(Z ≤ −0.89) = 1 − Φ(0.89) = 1 − 0.8133 = 0.1867
Interpretation: There is approximately an 18.7% probability that 40 or fewer items are defective. Since the expected number is 45, this represents a below-average defect count.
(c)Find P(35 ≤ X ≤ 55) and explain what this interval represents in terms of standard deviations from the mean. 45 marks

Worked Solution

Z1 = (35−45)/5.612 = −10/5.612 ≈ −1.78
Z2 = (55−45)/5.612 = 10/5.612 ≈ 1.78
P = Φ(1.78) − Φ(−1.78) = 2Φ(1.78) − 1 = 2(0.9625) − 1 = 0.9250
Interpretation: The interval [35, 55] spans ±1.78 standard deviations from the expected defect count of 45. There is a 92.5% probability the number of defectives falls in this range — closely approaching the “95% = ±2σ” rule.
