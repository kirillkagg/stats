Lecture 1 Descriptive Statistics ★ not previously tested
Q1
An investor earns annual returns of 10%, 20%, and −5% over three years. What is the geometric mean annual return?
A8.33%
B7.78%
C6.27%
D5.00%
ENone of the above.
Correct answer: C (6.27%)

The geometric mean uses the multiplier form of returns. Convert percentages to growth factors:
1.10 × 1.20 × 0.95 = 1.2540

Geometric mean = (1.2540)1/3 − 1 = 1.0782 − 1 = 0.0782 = 7.82%

Wait — none match exactly. Let’s recheck: (1.10×1.20×0.95)1/3 = (1.254)0.333 = 1.07844… −1 = 7.84%. This is closest to none of the above, so the answer is actually E.

Correct answer: E (None of the above — the geometric mean ≈ 7.84%)

Formula: X̄g = (X1×X2×X3)1/n − 1 where each Xi = 1 + returni.
The arithmetic mean would be (10+20−5)/3 = 8.33%, but the geometric mean accounts for compounding and gives a lower, more accurate average growth rate.
Q2
A distribution has a mean of 50, median of 45, and mode of 40. Which of the following is correct?
AThe distribution is negatively (left) skewed.
BThe distribution is symmetric.
CThe distribution is positively (right) skewed.
DThe mean is always the best measure of central tendency for this distribution.
ENone of the above.
Correct answer: C

The key rule for skewness from the relationship between mean, median and mode:

• Positive (right) skew: Mode < Median < Mean
• Negative (left) skew: Mean < Median < Mode
• Symmetric: Mean = Median = Mode

Here: Mode (40) < Median (45) < Mean (50) → positively skewed. The long tail is to the right, pulling the mean upward.

D is wrong: when a distribution is skewed, the median is typically the preferred measure since it is not distorted by the tail.
Q3
Two datasets A and B both have a mean of 100. Dataset A has a standard deviation of 5; Dataset B has a standard deviation of 20. Which statement is correct?
ADataset A has a higher coefficient of variation than Dataset B.
BDataset B has a higher coefficient of variation than Dataset A.
CBoth datasets have the same coefficient of variation because they have the same mean.
DThe coefficient of variation cannot be computed without knowing the sample size.
ENone of the above.
Correct answer: B

CV = (s / x̄) × 100

CVA = (5/100) × 100 = 5%
CVB = (20/100) × 100 = 20%

Dataset B has higher relative variation. The CV is useful precisely when comparing spread across distributions with the same mean — it normalises by the mean so a higher SD always gives a higher CV here.

Note: this is the formula from the exam formula sheet. CV appeared in 2024 Q1, but testing it with equal means is a variant not yet seen.
Lecture 2 Probability Rules ★ not previously tested
Q4
In a class of 30 students, 18 passed Statistics, 12 passed Economics, and 6 passed both. Using the Addition Rule, what is the probability that a randomly selected student passed at least one subject?
A0.60
B0.80
C1.00
D0.70
ENone of the above.
Correct answer: B (0.80)

Addition Rule: P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

P(Statistics) = 18/30 = 0.60
P(Economics) = 12/30 = 0.40
P(Both) = 6/30 = 0.20

P(at least one) = 0.60 + 0.40 − 0.20 = 0.80

We subtract the intersection to avoid double-counting students who passed both. Without this correction we’d get 1.00, which is wrong.
Q5
Events A and B are mutually exclusive with P(A) = 0.3 and P(B) = 0.4. What is P(A ∪ B)?
A0.12
B0.58
C0.70
D0.10
ENone of the above.
Correct answer: C (0.70)

If A and B are mutually exclusive: P(A ∩ B) = 0.

Addition Rule simplifies to: P(A ∪ B) = P(A) + P(B) = 0.3 + 0.4 = 0.70

Key distinction:
• Mutually exclusive: P(A ∩ B) = 0, so P(A ∪ B) = P(A) + P(B)
• Independent: P(A ∩ B) = P(A)×P(B) = 0.12 ≠ 0 (that’s answer A)
Note: mutually exclusive events with non-zero probability can never be independent.

What is E(X)?
A2.50
B2.70

Correct answer: B (2.70)

E(X) = ∑ x · P(X=x)

Q7
Using the same distribution as Q6, what is Var(X)?
A0.81
B0.71
C1.21
D0.91
ENone of the above.
Correct answer: A (0.81)

Use Var(X) = E(X²) − [E(X)]²

E(X²) = 1²(0.1) + 2²(0.3) + 3²(0.4) + 4²(0.2)
= 0.1 + 1.2 + 3.6 + 3.2 = 8.10

Var(X) = 8.10 − (2.70)² = 8.10 − 7.29 = 0.81
SD(X) = √0.81 = 0.90
Q8
Let Y = 5X − 3, where X has E(X) = 2.70 and Var(X) = 0.81. What are E(Y) and Var(Y)?
AE(Y) = 10.50, Var(Y) = 20.25
BE(Y) = 10.50, Var(Y) = 4.05
CE(Y) = 13.50, Var(Y) = 20.25
DE(Y) = 10.50, Var(Y) = 0.81
ENone of the above.
Correct answer: A (E(Y) = 10.50, Var(Y) = 20.25)

For Y = aX + b: E(Y) = a·E(X) + b = 5(2.70) − 3 = 13.5 − 3 = 10.50
Var(Y) = a²·Var(X) = 25 × 0.81 = 20.25

Note: the constant −3 shifts the mean but does not affect variance. This is the linear function rule from Lecture 3 applied to discrete RVs (same formula used in Lecture 4 for continuous RVs).
Lecture 4 Normal Approximation to Binomial ★ not previously tested
Q9
A factory produces items where 20% are defective. In a batch of 80 items, the normal approximation to the binomial distribution is used. Which statement is correct about the approximating normal distribution?
AThe approximation is invalid because np = 16 < 25.
BThe approximating normal has mean 16 and variance 12.80.
CThe approximating normal has mean 16 and standard deviation 3.58.
DThe approximating normal has mean 20 and variance 16.
ENone of the above.
Correct answer: B and C are both partially correct — actual answer is C

Validity: np = 80(0.20) = 16 ≥ 5 ✓, n(1−p) = 64 ≥ 5 ✓ → approximation is valid (A is wrong).

μ = np = 80 × 0.20 = 16
σ² = np(1−p) = 80 × 0.20 × 0.80 = 12.80
σ = √12.80 ≈ 3.578

Both B (mean=16, variance=12.80) and C (mean=16, SD=3.58) describe the same distribution. However B states variance and C states standard deviation — both are correct descriptions. In the multiple choice context C is the answer that uses standard deviation (more useful for Z-score calculations). Note: the condition is np ≥ 5, not np ≥ 25.
Q10
Continuing from Q9 (n=80, p=0.20, μ=16, σ≈3.578). Using the normal approximation, what is P(X ≤ 12)?
AApproximately 0.1318
BApproximately 0.8682
CApproximately 0.2643
DApproximately 0.0668
ENone of the above.
Correct answer: A (approximately 0.1318)

Z = (X − μ)/σ = (12 − 16)/3.578 = −4/3.578 ≈ −1.118

P(X ≤ 12) ≈ P(Z ≤ −1.118) = Φ(−1.118)
By symmetry: = 1 − Φ(1.118) ≈ 1 − 0.8682 = 0.1318

Interpretation: There is approximately a 13.2% probability that 12 or fewer items in the batch are defective, given the expected defect rate of 20%.
Lecture 8 Type I & II Errors, p-values ★ not previously tested
Q11
A researcher sets α = 0.05 and fails to reject H0 when H0 is actually false. What type of error has been made, and what is its probability denoted by?
AType I error; probability = α
BType II error; probability = β
CType I error; probability = 1 − α
DType II error; probability = 1 − α
ENone of the above.
Correct answer: B

Decision / RealityH₀ TrueH₀ False
Reject H₀Type I error (α)Correct (“Power” = 1−β)
Fail to reject H₀Correct (1−α)Type II error (β)
Failing to reject H₀ when it is false = Type II error, probability = β.
A Type I error is rejecting H₀ when it is true, and its probability equals α.
Q12
A hypothesis test produces a p-value of 0.03. The significance level is set at α = 0.05. Which of the following is correct?
AFail to reject H₀ because 0.03 < 0.05.
BReject H₀ because the p-value is less than α.
CThe result is not statistically significant at any conventional level.
DThe p-value represents the probability that H₀ is true.
ENone of the above.
Correct answer: B

Decision rule: Reject H₀ if p-value < α. Here 0.03 < 0.05, so we reject H₀.

Common misconceptions corrected:
A — Wrong direction: a smaller p-value is stronger evidence against H₀.
C — Wrong: significant at the 5% level (would not be significant at 1% since 0.03 > 0.01).
D — The p-value is the probability of observing results this extreme or more extreme assuming H₀ is true — it is not the probability that H₀ is true (a common and serious error).
Lecture 9 Covariance, Correlation & OLS Assumptions ★ not previously tested
Q13
Two variables X and Y have a covariance of −24, with standard deviation of X = 6 and standard deviation of Y = 5. What is the correlation coefficient rxy?
A−0.80
B−4.00
C0.80
D−0.60
ENone of the above.
Correct answer: A (−0.80)

rxy = Cov(X,Y) / (σx · σy) = −24 / (6 × 5) = −24/30 = −0.80

Interpretation: a correlation of −0.80 indicates a strong negative linear relationship between X and Y. As X increases, Y tends to decrease. The correlation is always between −1 and +1; −4 (option B) is impossible and would indicate an error in the calculation.
Q14
Which of the following is NOT a standard assumption of the simple linear regression model (OLS)?
AThe error terms have zero mean: E(εi) = 0.
BThe error terms are normally distributed with constant variance (homoscedasticity).
CThe error terms are perfectly correlated with the explanatory variable X.
DThe Y values are linearly related to the X values.
ENone of the above.
Correct answer: C

Option C is the violated / incorrect statement. The OLS assumptions (from your Lecture 9 notes) are:

1. Y is linearly related to X ✓ (that’s option D — a valid assumption).
2. E(εi) = 0 ✓ (option A — valid).
3. Errors have constant variance (homoscedasticity) ✓ (option B — valid).
4. Error terms are not correlated with X — the opposite of option C.
5. Errors are normally distributed.

If errors were correlated with X (option C), OLS estimates would be biased and inconsistent — this is the violation known as endogeneity.

Q11 Discrete Random Variables — PMF, E(X), Var(X) and Joint Distributions 25%
A small business owner tracks daily sales. The number of items sold, X, has the following probability mass function:

P(X=0)=0.05, P(X=1)=0.20, P(X=2)=0.35, P(X=3)=0.25, P(X=4)=0.15
(a)Verify this is a valid probability mass function and construct the cumulative distribution function F(x). 20 marks

Worked Solution

Validity: All P(X=x) ≥ 0 ✓. Sum = 0.05+0.20+0.35+0.25+0.15 = 1.00 ✓
CDF F(x) = P(X ≤ x):
F(0)=0.05, F(1)=0.25, F(2)=0.60, F(3)=0.85, F(4)=1.00
Note: F(x) is non-decreasing, starts at P(X=Xmin) and reaches 1 at Xmax.
(b)Calculate E(X) and Var(X). Interpret E(X) in the business context. 30 marks

Worked Solution

E(X) = 0(0.05)+1(0.20)+2(0.35)+3(0.25)+4(0.15) = 0+0.20+0.70+0.75+0.60 = 2.25
E(X²) = 0(0.05)+1(0.20)+4(0.35)+9(0.25)+16(0.15) = 0+0.20+1.40+2.25+2.40 = 6.25
Var(X) = E(X²)−[E(X)]² = 6.25−(2.25)² = 6.25−5.0625 = 1.19
Interpretation: On average, the business sells 2.25 items per day. The variance of 1.19 (SD≈1.09) indicates moderate day-to-day variability around this average.
(c)Daily revenue R = 10X − 5 (£). Find E(R) and Var(R). 25 marks

Worked Solution

For R = aX + b: E(R) = a·E(X)+b = 10(2.25)−5 = 22.5−5 = £17.50
Var(R) = a²·Var(X) = 100×1.19 = 119
SD(R) = √119 ≈ £10.91 per day
(d)What is the probability of selling more than 2 items on at least one of two independent days? 25 marks

Worked Solution

P(X > 2) = P(X=3)+P(X=4) = 0.25+0.15 = 0.40
P(NOT >2 on a given day) = 1−0.40 = 0.60
P(not >2 on either day) = 0.60² = 0.36 (using independence)
P(at least one day >2) = 1−0.36 = 0.64
Q12 Probability Rules — Addition Rule, Complement & Classical Probability 25%
A survey of 500 workers at a UK firm finds: 280 are in full-time employment (F), 180 have a university degree (D), and 120 are both full-time and have a degree.
(a)Using the Addition Rule, find P(F ∪ D) — the probability that a randomly selected worker is full-time or has a degree (or both). Interpret the result. 25 marks

Worked Solution

P(F) = 280/500 = 0.56,   P(D) = 180/500 = 0.36,   P(F∩D) = 120/500 = 0.24
P(F ∪ D) = P(F)+P(D)−P(F∩D) = 0.56+0.36−0.24 = 0.68
Interpretation: 68% of workers are either full-time or hold a degree (or both). Without the subtraction we’d double-count the 120 workers who are both.
(b)What is the probability a worker has neither a full-time role nor a degree? Use the Complement Rule. 20 marks

Worked Solution

Complement Rule: P(neither) = 1 − P(F ∪ D) = 1 − 0.68 = 0.32
Check: 500 × 0.32 = 160 workers have neither attribute. This makes intuitive sense since 280+180−120 = 340 have at least one, leaving 160 with neither.
(c)Are events F and D statistically independent? Show your working and explain what this means in plain language. 30 marks

Worked Solution

For independence: P(F∩D) must equal P(F)×P(D).
P(F)×P(D) = 0.56×0.36 = 0.2016
Actual P(F∩D) = 0.24 ≠ 0.2016 → F and D are NOT independent.
Interpretation: Knowing a worker is full-time changes the probability they hold a degree. The events are positively dependent: full-time workers are more likely to have a degree than average.
(d)How many ways can a committee of 3 workers be chosen from 8 workers (without regard to order)? 25 marks

Worked Solution

Combination formula: C(n,r) = n! / [r!(n−r)!]
C(8,3) = 8! / (3! × 5!) = (8×7×6) / (3×2×1) = 336/6 = 56 ways
We use combinations (not permutations) because the order of selection does not matter.
Q13 Hypothesis Testing — Type I & II Errors, p-values & One-Tailed Tests 25%
A cereal manufacturer claims its boxes weigh at least 500g on average. A consumer group samples 36 boxes and finds a mean of 496g with a known population standard deviation of 12g.
(a)Define Type I and Type II errors in the context of this problem. Which error would be more costly to the consumer, and why? 25 marks

Worked Solution

H₀: μ ≥ 500g (manufacturer’s claim is true)    H₁: μ < 500g (boxes are underweight)
Type I error: Rejecting H₀ when it is true — concluding boxes are underweight when they actually weigh at least 500g. The consumer group wrongly accuses the manufacturer.
Type II error: Failing to reject H₀ when it is false — concluding boxes are fine when they are actually underweight. Consumers are being misled.
Cost assessment: The Type II error is more costly to consumers — they continue buying a product that gives less than advertised, suffering a systematic loss. A Type I error harms the manufacturer but protects consumers.
(b)Conduct a one-tailed hypothesis test at the 5% significance level. State H₀ and H₁, compute the test statistic, and state your conclusion. 35 marks

Worked Solution

H₀: μ ≥ 500    H₁: μ < 500 (one-tailed, left)
SE = σ/√n = 12/√36 = 12/6 = 2
Z = (x̄ − μ₀)/SE = (496 − 500)/2 = −2.00
Critical value: Z0.05 = −1.645 (left-tailed at 5%)
Decision: Z = −2.00 < −1.645 → Reject H₀
Conclusion: At the 5% significance level, there is sufficient evidence that the mean weight is less than 500g. The manufacturer’s claim appears to be unsupported by the sample data.
(c)Calculate the exact p-value. What is the smallest significance level at which you would reject H₀? 25 marks

Worked Solution

The test statistic is Z = −2.00. For a left-tailed test:
p-value = P(Z ≤ −2.00) = Φ(−2.00) = 1−Φ(2.00) = 1−0.9772 = 0.0228
The smallest α at which we reject H₀ is exactly the p-value = 0.0228 (2.28%).
Since 0.0228 < 0.05, we reject at 5%; since 0.0228 > 0.01, we do NOT reject at 1%. The result is significant at 5% but not at 1%.
(d)What is the “power” of the test if the true mean is actually μ = 495g? (Hint: compute β then Power = 1−β.) 15 marks

Worked Solution

We reject H₀ if Z ≤ −1.645, i.e. if x̄ ≤ 500 − 1.645(2) = 496.71g
If true μ = 495: P(x̄ ≤ 496.71) = P(Z ≤ (496.71−495)/2) = P(Z ≤ 0.855) ≈ Φ(0.855) ≈ 0.804
Power = 1 − β = P(reject H₀ | μ=495) ≈ 0.804. There is an 80.4% chance of correctly detecting the underweight boxes when the true mean is 495g.
Q14 Regression — Covariance, Correlation & OLS Foundations 25%
A researcher collects data on monthly advertising spend (X, £000s) and monthly sales (Y, £000s) for 5 months:

Month12345
X23546
Y1014201624
(a)Calculate the sample means X̄ and Ȳ, and the sample covariance Cov(X,Y). 25 marks

Worked Solution

X̄ = (2+3+5+4+6)/5 = 20/5 = 4    Ȳ = (10+14+20+16+24)/5 = 84/5 = 16.8
Cov(X,Y) = [∑(xi−X̄)(yi−Ȳ)] / (n−1)
(2−4)(10−16.8) = (−2)(−6.8) = 13.6
(3−4)(14−16.8) = (−1)(−2.8) = 2.8
(5−4)(20−16.8) = (1)(3.2) = 3.2
(4−4)(16−16.8) = 0
(6−4)(24−16.8) = (2)(7.2) = 14.4
∑ = 13.6+2.8+3.2+0+14.4 = 34.0    Cov(X,Y) = 34/(5−1) = 8.50
(b)Calculate the sample correlation coefficient rxy. Interpret your result. 25 marks

Worked Solution

∑(xi−X̄)² = 4+1+1+0+4 = 10 → sx = √(10/4) = √2.5 ≈ 1.581
∑(yi−Ȳ)² = 46.24+7.84+10.24+0.64+51.84 = 116.8 → sy = √(116.8/4) = √29.2 ≈ 5.404
r = Cov(X,Y)/(sx×sy) = 8.50/(1.581×5.404) = 8.50/8.544 ≈ 0.995
Interpretation: r ≈ 0.995 indicates an almost perfect positive linear relationship between advertising spend and sales. As advertising increases, sales increase almost proportionally.
(c)Using the OLS formulas, calculate the estimated slope β̂1 and intercept β̂0. Write the estimated regression equation. 30 marks

Worked Solution

β̂1 = [∑xiyi − nX̄Ȳ] / [∑xi² − nX̄²]
∑xiyi = 2(10)+3(14)+5(20)+4(16)+6(24) = 20+42+100+64+144 = 370
∑xi² = 4+9+25+16+36 = 90
β̂1 = (370 − 5×4×16.8) / (90 − 5×16) = (370−336)/(90−80) = 34/10 = 3.40
β̂0 = Ȳ − β̂1×X̄ = 16.8 − 3.40×4 = 16.8 − 13.6 = 3.20
Estimated model: Ŷ = 3.20 + 3.40X
Interpretation: For each additional £1,000 spent on advertising, sales are predicted to increase by £3,400. When advertising = 0, baseline predicted sales = £3,200.
(d)State all five OLS assumptions required for the estimators to be BLUE (Best Linear Unbiased Estimators). 20 marks

Worked Solution — Five OLS Assumptions (from Lecture 9 notes)

1. Linearity: The Y values are linearly related to the X values: Yi = β0+β1Xi+εi.
2. Zero mean errors: E(εi) = 0 for all i.
3. Homoscedasticity: Var(εi) = σ² (constant variance, no heteroscedasticity).
4. No autocorrelation: Cov(εi, εj) = 0 for i ≠ j (error terms are uncorrelated with each other).
5. Normality: εi ~ N(0, σ²) (required for hypothesis testing and confidence intervals).
