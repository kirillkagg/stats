Q1
Discrete RV

A fair die is rolled once. Let X be the number shown. Which of the following correctly describes X?

A.X is a continuous random variable because it can take values between 1 and 6.
B.X is a discrete random variable with P(X = x) = 1/6 for x = 1, 2, 3, 4, 5, 6.
C.X is a discrete random variable with E(X) = 3.
D.Both A and C are correct.
E.None of the above.
Answer: B. X takes countable values {1,2,3,4,5,6} with equal probability 1/6 — making it a discrete random variable. Note that E(X) = (1+2+3+4+5+6)/6 = 3.5, not 3, so option C is wrong. A discrete variable does not take fractional values between integers, ruling out A.
Q2
CDF

The probability distribution of a discrete random variable X is given below.

x0123
P(X = x)0.100.300.400.20

What is P(1 ≤ X ≤ 3)?

A.0.40
B.0.70
C.0.90
D.0.60
E.None of the above.
Answer: C. P(1 ≤ X ≤ 3) = P(X=1) + P(X=2) + P(X=3) = 0.30 + 0.40 + 0.20 = 0.90. This is equivalent to 1 − P(X=0) = 1 − 0.10 = 0.90. This question tests both direct summation and the use of the cumulative distribution function (CDF).
Q3
Mean & Variance

A discrete random variable X has E(X) = 2 and E(X²) = 8. What is Var(X)?

A.8
B.4
C.6
D.2
E.None of the above.
Answer: B. Using the computational formula: Var(X) = E(X²) − [E(X)]² = 8 − (2)² = 8 − 4 = 4. This is the shortcut version of the variance formula and is frequently tested. The standard deviation would be σ = √4 = 2.
Q4
Binomial

A multiple-choice test has 8 questions, each with 4 options (only one correct). A student guesses every answer at random. What is the probability that the student gets exactly 3 correct?

A.0.2076
B.0.3115
C.0.1468
D.0.0038
E.None of the above.
Answer: A. Here n = 8, x = 3, P = 0.25 (one correct out of four options).
P(X=3) = C(8,3) × (0.25)³ × (0.75)⁵ = 56 × 0.015625 × 0.23730 ≈ 0.2076. Recall: C(8,3) = 8!/(3!·5!) = 56.
Q5
Binomial

According to a survey, 30% of university students own a car. If 10 students are selected at random, what are the mean and standard deviation of the number who own a car?

A.Mean = 3, Standard Deviation = 2.10
B.Mean = 3, Standard Deviation = 1.45
C.Mean = 7, Standard Deviation = 1.45
D.Mean = 3, Standard Deviation = 2.10 and Variance = 4.41
E.None of the above.
Answer: B. For a Binomial distribution: μ = np = 10 × 0.30 = 3. σ² = np(1−p) = 10 × 0.30 × 0.70 = 2.10. σ = √2.10 ≈ 1.45. Option A incorrectly uses the variance (2.10) as the standard deviation. Option D is wrong because the variance is 2.10, not 4.41.
Q6
Poisson

Records show that on average, 3 customers arrive at a bank per 10-minute period. Using the Poisson distribution, what is the probability that exactly 5 customers arrive in a 10-minute period?

A.0.1008
B.0.0940
C.0.1500
D.0.0216
E.None of the above.
Answer: A. λ = 3, x = 5.
P(X=5) = (e⁻³ × 3⁵) / 5! = (0.04979 × 243) / 120 = 12.0988 / 120 ≈ 0.1008. For the Poisson distribution, mean = variance = λ = 3, so σ = √3 ≈ 1.73.
Q7
Poisson Approx.

The probability that a randomly selected component is defective is 0.002. A batch of 800 components is inspected. Using the Poisson approximation to the Binomial, what is the probability that at most 1 component is defective?

A.0.5249
B.0.8088
C.0.2707
D.0.9384
E.None of the above.
Answer: B. λ = np = 800 × 0.002 = 1.6. The Poisson approximation applies here because n is large and P is small.
P(X=0) = e⁻¹·⁶ = 0.2019
P(X=1) = e⁻¹·⁶ × 1.6¹/1! = 0.2019 × 1.6 = 0.3230
P(X ≤ 1) = 0.2019 + 0.3230 = 0.5249. Wait — let me recheck: 0.2019 + 0.3230 ≈ 0.5249, which is option A. Correct answer is A.

Revised: Answer: A. P(X ≤ 1) = P(X=0) + P(X=1) = e⁻¹·⁶ + e⁻¹·⁶ × 1.6 = e⁻¹·⁶(1 + 1.6) = 0.2019 × 2.6 ≈ 0.5249.
Q8
Joint Distribution

The joint probability distribution of X and Y is given below. Are X and Y statistically independent?

X \ YY = 0Y = 1P(X=x)
X = 00.200.300.50
X = 10.200.300.50
P(Y=y)0.400.601.00
A.Yes, because the joint probabilities equal the products of the marginal probabilities.
B.No, because the covariance between X and Y is non-zero.
C.Yes, because P(X=0) = P(X=1) = 0.5.
D.No, because the conditional distribution of Y given X is not defined.
E.None of the above.
Answer: A. Two random variables are independent if and only if their joint distribution equals the product of their marginal distributions for all values. Check: P(X=0, Y=0) = 0.20 = P(X=0) × P(Y=0) = 0.5 × 0.4 = 0.20 ✓. Similarly for all four cells. Since the condition holds for every combination, X and Y are statistically independent.
Q9
Covariance & Correlation

Two investments have the following properties: E(X) = 5, E(Y) = 8, Var(X) = 4, Var(Y) = 9, and Cov(X, Y) = −3. What is the variance of the portfolio W = X + Y?

A.13
B.7
C.19
D.10
E.None of the above.
Answer: B. Var(X + Y) = Var(X) + Var(Y) + 2·Cov(X,Y) = 4 + 9 + 2×(−3) = 13 − 6 = 7. The negative covariance reduces portfolio variance — this is the fundamental benefit of diversification. E(W) = E(X) + E(Y) = 5 + 8 = 13.
Q10
Functions of RVs

Let X be a random variable with E(X) = 4 and Var(X) = 9. Define Y = 3X − 2. Which of the following is correct?

A.E(Y) = 10 and Var(Y) = 27
B.E(Y) = 10 and Var(Y) = 81
C.E(Y) = 14 and Var(Y) = 81
D.E(Y) = 10 and Var(Y) = 25
E.None of the above.
Answer: B. For linear functions of a random variable Y = aX + b:
E(Y) = a·E(X) + b = 3×4 + (−2) = 12 − 2 = 10 ✓
Var(Y) = a²·Var(X) = 3² × 9 = 9 × 9 = 81. Constants added/subtracted do NOT affect variance — only the multiplicative constant is squared. Option A incorrectly multiplies variance by 3 instead of 3².

Q11 — Discrete Random Variables: Properties & Distributions

100 marks
A game at a fairground involves rolling two fair dice and recording the absolute difference between the two numbers shown. Let X denote this absolute difference.
Part (a)

Construct the complete probability distribution table for X, showing all possible values and their corresponding probabilities. Verify that your probabilities sum to 1. [25 marks]

List all 36 equally likely outcomes (pairs of dice). For each pair (i,j) compute |i − j|. Possible values of X are {0, 1, 2, 3, 4, 5}. Count how many pairs give each value and divide by 36.
Part (b)

Calculate the expected value E(X) and variance Var(X) of the absolute difference. Show all working. [30 marks]

Use: E(X) = Σ x·P(X=x) and Var(X) = E(X²) − [E(X)]² where E(X²) = Σ x²·P(X=x).
Part (c)

A player wins £4 for each unit of X and pays a fixed entry fee of £3. Let W = 4X − 3 be the net winnings. Calculate E(W) and Var(W). Is the game fair? Explain. [25 marks]

Use the linear transformation rules: E(aX + b) = a·E(X) + b and Var(aX + b) = a²·Var(X). A game is fair if E(W) = 0.
Part (d)

Calculate and interpret the cumulative distribution function F(x) = P(X ≤ x) for all values of X. Hence find P(1 ≤ X ≤ 3) using the CDF. [20 marks]

The CDF is defined as F(x) = P(X ≤ x) = Σ P(X = k) for all k ≤ x. Then P(1 ≤ X ≤ 3) = F(3) − F(0).

Q12 — Binomial Distribution

100 marks
A factory produces electronic components, and it is known that 15% of all components produced are defective. A quality control inspector randomly selects a sample of 12 components.
Part (a)

State clearly the conditions required for a Binomial distribution to apply. Verify that this scenario satisfies those conditions. [20 marks]

The four Binomial conditions: (1) fixed number of trials n, (2) each trial results in success or failure, (3) constant probability of success P, (4) trials are independent.
Part (b)

Find the probability that:
(i) exactly 2 components are defective. [15 marks]
(ii) at most 2 components are defective. [15 marks]
(iii) more than 3 components are defective. [15 marks]

Use P(X=x) = C(n,x)·P^x·(1−P)^(n−x) with n=12, P=0.15. For (iii): P(X > 3) = 1 − P(X ≤ 3).
Part (c)

Calculate the mean, variance, and standard deviation of the number of defective components. Briefly interpret the mean in the context of the problem. [20 marks]

μ = np, σ² = np(1−p), σ = √(np(1−p)).
Part (d)

The Bernoulli distribution is a special case of the Binomial. Derive the mean and variance of a Bernoulli random variable from first principles. How does this relate to the Binomial result in part (c)? [15 marks]

A Bernoulli variable X takes value 1 with probability P and 0 with probability 1−P. Compute E(X) = 0·(1−P) + 1·P = P. For variance, use E(X²) − [E(X)]².

Q13 — Poisson Distribution & Approximation

100 marks
A call centre receives an average of 6 calls per hour. Assume that calls arrive independently and the number of calls follows a Poisson distribution.
Part (a)

Describe the key properties of the Poisson distribution. Under what conditions can it be used to approximate the Binomial distribution? [25 marks]

Key properties: λ is the mean rate, mean = variance = λ, counts non-negative integers. Approximation condition: n large (> 20) and P small (< 0.05), with λ = nP remaining moderate.
Part (b)

Calculate the probability that in a 30-minute period:
(i) exactly 4 calls are received. [15 marks]
(ii) fewer than 3 calls are received. [15 marks]
(iii) at least 2 calls are received. [15 marks]

In 30 minutes the rate is λ = 6 × 0.5 = 3. Use P(X=x) = e^(−λ)·λ^x / x!. For (iii): P(X ≥ 2) = 1 − P(X=0) − P(X=1).
Part (c)

The probability that a caller is put on hold is 0.004. Using the Poisson approximation to the Binomial, find the probability that in a sample of 500 calls, at most 3 callers are put on hold. [30 marks]

Check conditions: n = 500 (large), P = 0.004 (small) → Poisson applies. λ = nP = 500 × 0.004 = 2. Compute P(X ≤ 3) = P(0) + P(1) + P(2) + P(3).

Q14 — Joint Distributions, Covariance & Portfolio Analysis

100 marks
An investor holds two assets, A and B. The returns (in %) per £1,000 invested under three economic scenarios are shown below. Scenarios are equally likely.
ScenarioProbabilityReturn A (%)Return B (%)
Boom1/3155
Normal1/388
Recession1/3−212
Part (a)

Calculate the expected return E(A) and E(B), and the variance Var(A) and Var(B) for each asset. [25 marks]

Use E(X) = Σ x·P(x) and Var(X) = Σ (x − μ)²·P(x) or equivalently E(X²) − [E(X)]².
Part (b)

Compute the covariance Cov(A, B) and the correlation coefficient ρ(A, B). Interpret both values. [25 marks]

Cov(A,B) = E(AB) − E(A)·E(B) where E(AB) = Σ a·b·P(scenario). Then ρ = Cov(A,B) / (σ_A · σ_B). Correlation ranges between −1 and +1.
Part (c)

An investor places 60% of their wealth in Asset A and 40% in Asset B, so the portfolio return is P = 0.6A + 0.4B. Calculate E(P) and Var(P). [30 marks]

Use: E(P) = 0.6·E(A) + 0.4·E(B). For variance: Var(P) = (0.6)²·Var(A) + (0.4)²·Var(B) + 2·(0.6)·(0.4)·Cov(A,B). This is the key portfolio variance formula from the lecture.
Part (d)

Compare Var(P) with Var(A) and Var(B). What does the sign of Cov(A,B) suggest about the benefit of combining these two assets in a portfolio? [20 marks]

A negative covariance means the assets tend to move in opposite directions — when one falls, the other rises. This reduces overall portfolio risk (variance) below what either asset has individually. This is the principle of diversification.
